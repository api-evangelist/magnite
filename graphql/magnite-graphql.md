# Magnite GraphQL API

Magnite is an independent sell-side advertising platform (SSP). Their API covers publisher inventory management, deal creation, buyer targeting, bid requests, and reporting for display, video, and CTV inventory.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Magnite REST API
**Documentation:** https://help.magnite.com/help/developer-documentation
